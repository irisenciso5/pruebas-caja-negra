# Pruebas Unitarias de Caja Negra

Este proyecto contiene un módulo de biblioteca simple y pruebas unitarias de caja negra para verificar su funcionamiento externo.

## Cómo ejecutar las pruebas

1. Instalar las dependencias:
   ```
   npm install
   ```

2. Ejecutar todas las pruebas:
   ```
   npm run test:biblioteca
   ```
